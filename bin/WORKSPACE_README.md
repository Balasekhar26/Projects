# Balu Projects Workspace

This repository is organized as one workspace with one canonical top-level folder per active project:

- `universal-ai` - local AI desktop/backend system.
- `pcb-doctor` - board diagnostics and measurement tooling.
- `ai-cyber-shield` - defensive security agent and dashboard.
- `universal-translator` - canonical ULT / Universal Translator project path.
- `musical-keyboard` - instrument keyboard app experiments.
- `dews` - Defensive Early Warning System scaffold.
- `07-NeuroSeed` - memory reinforcement concept and prototype.

The old translator path `ult-translator/` is retired. Keep translator source under `universal-translator/` only.

Generated runtime data, virtual environments, vendor folders, model downloads, packaged app outputs, local databases, and recorded audio are ignored intentionally. Recreate those artifacts locally from each project's setup scripts instead of committing them.
