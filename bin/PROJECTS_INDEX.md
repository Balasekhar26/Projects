## Project 6: DEWS

Folder: `dews`

Status: safe simulation and dashboard scaffold stored

Original motive: DEWS began as a "weapon destroyer" / directed-energy weapon-system concept: AI cameras, thermal sensing, and other sensors would identify weapons or ammunition at a distance, then a remote energy or metal-heating mechanism would attempt to disable the threat.

Safe build direction: This repository must stay as a Defensive Early Warning System: detect hazards, correlate camera/thermal/sensor evidence, alert humans, preserve evidence, simulate safety cases, and recommend protective action. It must not include instructions, control logic, targeting logic, or hardware designs for damaging weapons, ammunition, people, devices, or infrastructure.

## Project 7: NeuroSeed

Folder: `07-NeuroSeed`

Status: concept dossier and prototype stored

Original motive: NeuroSeed began as an idea for wirelessly injecting information into the human brain to reduce education time, so people could spend more time understanding, practicing, and applying skills.

Summary: A science-grounded memory reinforcement project based on awake learning plus sleep-timed targeted memory reactivation cues. The project explicitly avoids unsupported claims about direct memory upload, memory copying, or unconscious information injection. A static prototype is available at `07-NeuroSeed/prototype/index.html`.
