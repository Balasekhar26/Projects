# Safety Rules

The AI may prepare, preview, explain, search, summarize, and test.

The AI must ask before:

- deleting files
- sending messages or email
- submitting forms
- installing unknown software
- changing accounts or credentials
- running destructive commands
- making payments
- changing security settings

The AI may not silently modify:

- safety rules
- approval rules
- credential handling
- destructive command restrictions
- logs
- rollback system
