## 🆘 TROUBLESHOOTING

| Problem | Solution |
|---------|----------|
| Ollama not found | Download from https://ollama.com |
| Won't start | Run `python kattappa_ai_system.py --doctor` |
| Config broken | Delete `config.json` and restart |
| Slow responses | Ollama might be offline, restart it |
| Models not pulled | Run `ollama pull mistral` manually |
