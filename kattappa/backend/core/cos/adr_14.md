# ADR-14: World Model Specification

*   **Status**: `ACCEPTED`
*   **Date**: 2026-06-29
*   **Evidence Level**: **E4** (Approved Blueprint)

### Context & Problem
Planners need to simulate actions and predict outcomes before execution. If prediction is combined directly with planning, it creates high coupling. Furthermore, rule-based simulation fails in complex, non-deterministic, or novel scenarios where explicit rules are unavailable.

### Decision
Define a partitioned **World Model** that separates transition predictions, reward calculation, and uncertainty metrics into isolated models.

---

### World Model Subsystem Architecture

```
                       World Model Interface
       ┌───────────────────┼────────────────────┐
       ▼                   ▼                    ▼
Transition Model      Reward Model      Uncertainty Model
  P(s' | s, a)          R(s, a)               σ(s)
```

#### 1. Transition Model: $P(s' \mid s, a)$
- Generates a probability distribution over potential next states $s'$ given the current state $s$ and proposed action $a$.
- Supports both symbolic transition laws and neural dynamics models for learned latent projections.

#### 2. Reward Model: $R(s, a)$
- Calculates the expected utility payoff or reward for executing action $a$ in state $s$.
- Incorporates penalty cost functions (compute budget, API usage, temporal constraints).

#### 3. Uncertainty Model: $\sigma(s)$
- Estimates variance or prediction uncertainty. High variance triggers meta-cognition to gather more information (run experiments or query users) before committing.

#### 4. Counterfactual Generator ("What-if" Sandbox)
- Creates sandbox branches (`WorldModelCoordinator` branches) to simulate alternative realities or alternative action paths without modifying active main beliefs.

---

### Predictive Processing Integration
The world model runs as a continuous predictor:
- Incoming observations are compared against predictions generated by $P(s' \mid s, a)$.
- The difference ($\text{Prediction Error}$) is calculated and backpropagated to refine transition weights and update belief states.
