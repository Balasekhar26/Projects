# Kattappa Cognitive OS Glossary

Precise definitions of terms to prevent semantic and architectural drift:

---

- **Belief**: An evolving, time-varying assertion stored with a confidence score, source metadata, and contradiction links. Differs from static facts in that beliefs can decay or update based on reality checks.
- **Fact**: An immutable verified node in Semantic Memory with high significance score ($\ge 0.50$).
- **Hypothesis**: A proposed causal relationship ("Doing X improves metric Y") generated by the Scientist Agent, verified through experiment loops.
- **World State**: The vector representation of environmental variables across the four domains: Physical, Social, Self, and Digital.
- **Transition Result**: The predicted state output of an action, containing confidence, uncertainty, assumptions, and unknowns.
- **Scratchpad**: Transient, fast key-value workspace registers for active sub-reasoning execution.
- **Sleep Buffer**: Staging database table (`sleep_buffer`) holding temporary observations before significance-based promotion.
- **Confidence**: Probability (0.0 to 1.0) indicating system belief strength.
- **Uncertainty**: Metric (0.0 to 1.0) representing information entropy, growing with prediction step count.
