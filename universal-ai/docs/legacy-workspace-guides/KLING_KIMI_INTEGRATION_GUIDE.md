## 🔧 **Setup Instructions**

### **Prerequisites:**
- Kling AI 2.1 API key
- Kimi AI API key
- Node.js 16+ installed
- FFmpeg for video processing

### **Installation:**
```bash
# Universal AI System
cd projects/universal-ai-system/integrations/kling-ai
npm install
npm start

cd projects/universal-ai-system/integrations/kimi-ai
npm install
npm start

# ULT Translator
cd projects/universal-translator/integrations/kimi-ai
npm install
npm start
```

### **Configuration:**
```bash
# Set environment variables
KLING_API_KEY=your_kling_api_key
KIMI_API_KEY=your_kimi_api_key
```
