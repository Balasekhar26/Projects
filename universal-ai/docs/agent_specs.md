# Agent Specs

- Planner: classifies tasks and selects agent route.
- Memory Agent: retrieves relevant Chroma/SQLite context.
- Safety Agent: blocks or pauses risky requests.
- Research Agent: reads web content and summarizes.
- Browser Agent: browser reading and search.
- Desktop Agent: gated screen/desktop control.
- Vision Agent: screenshot/OCR and visual explanation.
- Voice Agent: push-to-talk and speech output hooks.
- Coding Agent: coding plans, Git awareness, test runner hooks.
- Terminal Agent: allowlisted command execution and approval gating.
- File Agent: read/preview/write-with-approval flow.
- Evaluator Agent: final result and memory write-back.
- Self-Improvement Agent: proposes improvements, tests, and rollback plan.
